Network Flood Simulation

Networking Simulation

Simula un ataque de Denegación de Servicio Distribuido (DDoS) dirigido al localhost (127.0.0.1). Utiliza múltiples hilos para generar solicitudes HTTP en un entorno controlado. Está diseñado exclusivamente para fines educativos, permitiendo a los usuarios explorar conceptos de ciberseguridad sin afectar sistemas externos.

Dos Stealthy

Traffic Analysis Simulation

Este script crea una simulación breve de ataque DoS enviando tráfico a localhost. Con múltiples hilos, envía peticiones constantes durante 30 segundos de manera aleatoria, el script añade un mensaje secreto en algunas de las solicitudes, brindando así la oportunidad de capturarlo con herramientas de monitoreo de red como Wireshark para fines de análisis y práctica en un entorno seguro.
